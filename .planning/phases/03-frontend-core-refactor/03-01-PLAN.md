1. Create `src/lib/api.ts` — typed fetch wrapper:
   ```typescript
   import type { AnalysisResult, HistoryEntry, SaveChunkRequest } from "@/src/types/index";

   async function apiFetch<T>(url: string, options?: RequestInit): Promise<T> {
     const res = await fetch(url, options);
     if (!res.ok) {
       const body = await res.json().catch(() => ({ error: "Unknown error" }));
       throw new Error(body.error ?? `HTTP ${res.status}`);
     }
     return res.json() as Promise<T>;
   }
   ```
   Then export an `api` object with methods for all 8 API endpoints. Use project import style with `@/src/` prefix and double quotes. Key methods:
   - `checkSentence(sentence: string)` returns `Promise<AnalysisResult & { id: number }>`
   - `saveSentence(sentence: string, analysis: AnalysisResult)` returns `Promise<{ success: boolean; id: number }>`
   - `getAnalysisById(id: number)` returns `Promise<AnalysisResult & { id: number; sentence: string }>`
   - `getHistory()` returns `Promise<HistoryEntry[]>`
   - `getSaved()` returns `Promise<unknown[]>` (saved sentences shape is loosely typed in current codebase)
   - `getChunks()` returns `Promise<unknown[]>`
   - `saveSentenceToLibrary(sentence: string)` returns `Promise<{ success: boolean }>`
   - `saveChunk(chunk: SaveChunkRequest)` returns `Promise<{ success: boolean }>`

   - Export `historyQueryOptions()`, `savedSentencesQueryOptions()`, `chunksQueryOptions()` with default staleTime
   - Import from `@/src/lib/api` and use `@tanstack/react-query` `queryOptions` helper
